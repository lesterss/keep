#Шпаргалка команд Magento2


###Реиндекс
Список всех возможных реиндексов 
``` bash
php bin/magento indexer:info
```
Статус реиндекса
```bash
php bin/magento indexer:status
```
Реиндекс всего
``` bash
php bin/magento indexer:reindex
```
или по нейму
```bash
php bin\magento indexer:reindex indexer_name.
```
